​ 1.正向索引，跟数组一样，第一个元素从0开始

3.列表的常用操作方法

​ 查询下标：列表.index(元素) 若该元素不存在则会报错

​ 插入元素：列表.insert(下标，元素)

​ 追加元素：列表.append(元素) 该元素追加到列表尾部

​ 追加一批元素：列表.extend(其他列表) 将该列表追加到原本列表的尾部

​ 删除元素：第一种，del 列表[下标] 第二种，列表.pop(下标)，该方法能 返回删除的元素

​ 删除某元素在列表中的第一项：列表.remove(元素)

​ 清空列表：列表.clear()

​ 统计某元素在列表中的数量：列表.count(元素)

​ 统计列表内有多少个元素：len(列表)

7.集合的定义和操作

​ 定义一个空集合：集合 = set()

​ 不可重复，自动去重 无序，不支持下标索引

​ 添加元素：集合.add(元素)

​ 移除元素：集合.remove(元素)

​ 随取出一个元素：集合.pop() 取出该元素后，集合将不再包含被取 出的元素

​ 清空集合：集合.clear()

​ 统计集合元素数量：len(集合)

​ 集合不支持下标索引，不能用while循环，可以用for循环

14.文件的写出操作

​ w 操作：文件不存在会创建文件，文件存在会清空文件内容，写入我们想写的

​ write()写入内容，flush()刷新内容到硬盘中

​ close()方法自带flush()的功能

​ a 操作：文件不存在会创建文件，文件存在会再最后写入我们想写的

15.异常捕获

​ try: 可能出现异常的

​ except: 出现异常时执行的

​ else: 没出现异常时执行的

​ finally: 有无异常时都执行的

16.自定义模块并导入

​ _ _main__变量的功能是，只有当程序是直接执行的才会进入main下的代码，如果是 被导入的，则不执行

​ 不同模块的同名功能如果都被导入，后者将会覆盖前者

​ _ _all__变量可以控制 import * 时哪些功能是可以被导入的

17.JSON数据格式的转换

​ JSON就是一个字典或内部元素都是字典的列表

​ data = json.dumps(data) 将python数据转化为 json数据，如果有中文可以带上：ensure_ascii = False 参数来确保中文正常转换

​ josn.loads(data) 将json数据转化为python列表或 字典
